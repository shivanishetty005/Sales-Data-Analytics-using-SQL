# 📊 SQL Sales Data Analysis

## 📖 Overview
This project demonstrates SQL-based analysis on a sample sales dataset to extract meaningful business insights. It includes database exploration, date range analysis, key business metrics, product segmentation, and ranking reports using SQL queries.

---

## 🎯 Objectives
- Explore the database structure and available tables.
- Analyze the available sales data over time.
- Calculate important business metrics.
- Segment products based on cost.
- Rank products and customers based on performance.

---

## 🛠️ Technologies Used
- SQL
- SQL Server

---

- SELECT
- WHERE

- ORDER BY
- JOIN
- Common Table Expressions (CTEs)
- CASE Expressions
- Aggregate Functions (`SUM`, `COUNT`, `AVG`)
- Date Functions (`MIN`, `MAX`, `DATEDIFF`)

## 📌 Analysis Performed

### 🗂️ Database Exploration
- Retrieved database tables and schemas.
- Examined table columns and metadata.

### 📅 Data Range Analysis
- Identified the earliest and latest order dates.
- Calculated the available sales period.
- Determined customer age boundaries.

### 📦 Data Segmentation
- Categorized products into different cost ranges.
- Counted the number of products in each category.

### 📈 Metrics Analysis
- Calculated total sales revenue.
- Measured total quantity sold.
- Computed average selling price.
- Counted orders, products, and customers.
- Generated a business metrics summary report.

### 🏆 Ranking Analysis
- Retrieved the top-performing products by revenue.
- Identified the lowest-performing products.
- Ranked customers based on revenue.
- Found customers with the fewest orders.
